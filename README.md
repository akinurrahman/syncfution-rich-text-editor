# Rich Text Editor Project

## Overview
This project is a simple rich text editor application created using the Syncfusion library. It serves as a reference for integrating and using Syncfusion's RichTextEditorComponent in a React application.

## Syncfusion RichTextEditor Installation
```
npm install @syncfusion/ej2-react-richtexteditor --
```

## Setting Syncfusion License Key
i. Obtain a license key from Syncfusion for using their components. <br/>
ii. Set the environment variable SYNCFUSION_LICENSE_KEY with your license key.

```
SYNCFUTION_LICENCE_KEY="YOUR_LICENCE_KEY"
```
